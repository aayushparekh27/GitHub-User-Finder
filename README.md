# 🔍 GitHub User Finder Pro

An advanced and interactive **GitHub User Finder** web application that allows users to search any GitHub profile and view detailed analytics such as repositories, followers, activity timeline, and export options — all with a modern UI/UX.

---

## 🚀 Features

✨ **User Profile Analytics**
- GitHub avatar, name, username, bio, location
- Public repositories, followers & following count

📦 **Repository Insights**
- Recently updated repositories
- Language detection with color indicators
- Stars, forks, and open issues count

🕒 **Recent Activity Timeline**
- Push events, repo creation, stars, forks, etc.

🎨 **Modern UI / UX**
- Glassmorphism design
- Animated background & particles
- 3D flip profile card
- Skeleton loaders

🌙 **Dark / Light Mode**
- Theme toggle with persistent settings

📤 **Export & Share**
- Export profile data as **JSON**
- Export profile summary as **Text**
- Download profile **Screenshot**
- Share or copy GitHub profile link
